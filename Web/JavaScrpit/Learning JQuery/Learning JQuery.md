
## 概述
JQuery是一个“写的更少，但做的更多”的轻量级 JavaScript 库。

jQuery 库包含以下特性：

- HTML 元素选取
- HTML 元素操作
- CSS 操作
- HTML 事件函数
- JavaScript 特效和动画
- HTML DOM 遍历和修改
- AJAX
- Utilities


### 加载JQuery库
JQuery库需要从网上下载，就是一个.js文件。然后在header中引入。

```javascript
<head>
<script type="text/javascript" src="jquery.js"></script>
</head>
```
注意，`<script>` 标签应该位于页面的 `<head>` 部分。
